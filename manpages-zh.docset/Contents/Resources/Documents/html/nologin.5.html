NAME

nologin - 阻止非root用户登录系统  

描述 DESCRIPTION

如果存在文件 /etc/nologin， login(1) 将只允许root访问。其它用户的登录会遭到拒绝并且显示该文件中的内容给他们。  

文件 FILES

/etc/nologin  

参见 SEE ALSO

login(1), shutdown(8)
